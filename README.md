# EquaMate
A minimal desktop application for solving mathematical problems using numerical methods.

## available methods
✅ Linear systems
1.  Gauss
2.  Crammer
3.  LU
4.  LLt (Cholesky decomposition)
   
✅ Non-linear systems
1.  Newton Raphson
2.  Fixed point
   
✅ Root finding
1.  Secant
